Análisis de Spotify

Análisis exhaustivo de las tendencias de los artistas más escuchados en Spotify utilizando Python. El objetivo principal del proyecto fue identificar patrones en la popularidad de los artistas y cómo estos patrones evolucionan a lo largo del tiempo.

Análisis de Instagram

A diferencia del proyecto anterior, acá decidí centrarme en el análisis de Instagram. El objetivo principal del proyecto fue identificar patrones y tendencias en la popularidad de los influencers, así como entender cómo su impacto varía según diferentes factores como el tipo de contenido y la demografía de sus seguidores.

Visita el proyecto aquí

Herramientas

  • Pandas para la manipulación y limpieza de datos.
  • NumPy para cálculos numéricos avanzados.
  • Matplotlib y Seaborn para la visualización gráfica de las tendencias y patrones encontrados.
  • Spotipy, una biblioteca de Python para interactuar con la API de Spotify, lo que facilitó la extracción de datos relevantes.
